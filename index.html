Reproducible Biometric Evaluations

OpenBR is an end-to-end system for conduting biometric evaluations in an accessible, reproducible, and transparent manner. OpenBR supports both the NIST FRVT 2012 and 2006 APIs.

Development

OpenBR is licensed under Apache 2.0 and follows the Semantic Versioning convention for publishing releases. The project uses the CMake build system and depends on Qt 4.8 and OpenCV 2.4.3.
