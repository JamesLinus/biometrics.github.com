About

OpenBR is a collaborative research project started by The MITRE Corporation. The MITRE Corporation is a not-for-profit organization chartered to work in the public interest.

Development

OpenBR is supported on Windows, Mac OS X, and Debian Linux. The project is licensed under Apache 2.0 and published releases follow the Semantic Versioning convention. Internally the code base uses the CMake build system and has external dependencies on Qt 5.0 and OpenCV 2.4.3.
