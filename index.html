Reproducible Biometric Evaluations

OpenBR is an end-to-end system for running biometric evaluations in an accessible, reproducible, and transparent way. OpenBR supports both the NIST FRVT 2012 and 2006 APIs.

OpenBR - Opensource Biometrics Recognition and Evaluation

Development

OpenBR is written in C/C++ and follows the Semantic Versioning convention for publishing releases. The project uses the CMake build system and depends on Qt 4.8 and OpenCV 2.4.3. OpenBR supports both the NIST FRVT 2012 and 2006 APIs.
