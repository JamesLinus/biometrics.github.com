Windows 7 - Visual Studio Express Edition 2012 - x64
  1. Download Visual Studio 2012 Express Edition for Windows Desktop and install.
    1. Consider the free open source program WinCDEmu if you need a program to mount ISO images.
    2. You will have to register with Microsoft after installation, but it's free.
    3. Grab any available Visual Studio Updates.
  2. Download CMake 2.8.10.2 and install.
    1. During installation setup select "add CMake to PATH".
  3. Download OpenCV 2.4.3 and unarchive.
    1. Consider the free open source program 7-Zip if you need a program to unarchive tarballs.
    2. Move the "OpenCV-2.4.3" folder to "C:\".
    3. In "C:\OpenCV-2.4.3\modules\objdetect\src\haar.cpp" line 55, change "#ifdev CV_AVX" to "#ifdef 0".
    4. Open "VS2012 x64 Cross Tools Command Prompt" (from the Start Menu, select "All Programs" -> "Microsoft Visual Studio 2012" -> "Visual Studio Tools" -> "VS2012 x64 Cross Tools Command Prompt") and enter:
      $ cd C:\OpenCV-2.4.3
      $ mkdir build-msvc2012
      $ cd build-msvc2012
      $ cmake -G "NMake Makefiles" -D BUILD_PERF_TESTS=OFF -D BUILD_TESTS=OFF -D WITH_FFMPEG=OFF -D CMAKE_BUILD_TYPE=Debug ..
      $ nmake
      $ nmake install
      $ cmake -D CMAKE_BUILD_TYPE=Release ..
      $ nmake
      $ nmake install
      $ nmake clean
  4. Download Qt 5.0.1 and unzip.
    1. Install Perl/Python/Ruby dependencies as explained in the "Windows" section of "README". Make sure they are added to "path" when given the option during installation.
    2. Download Direct X Software Developement Kit and install.
    3. From the VS2012 x64 Cross Tools Command Prompt:
      $ cd qt-everywhere-opensource-src-5.0.1
      $ configure -prefix C:\Qt\5.0.1\msvc2012 -opensource -confirm-license -nomake examples -nomake tests
      $ nmake
      $ nmake install
      $ cd ..
      $ rmdir /Q /S qt-everywhere-opensource-src-5.0.1
    4. nmake will take several hours to finish.
  5. Create a GitHub account and follow their instructions for setting up Git.
    1. Launch "Git Bash" from the Desktop and clone OpenBR:
      $ cd /c
      $ git clone https://github.com/biometrics/openbr.git
      $ cd openbr
      $ git submodule init
      $ git submodule update
  6. Build OpenBR!
    1. From the VS2012 x64 Cross Tools Command Prompt:
      $ cd C:\openbr
      $ mkdir build-msvc2012
      $ cd build-msvc2012
      $ cmake -G "CodeBlocks - NMake Makefiles" -D CMAKE_PREFIX_PATH="C:/OpenCV-2.4.3/build-msvc2012/install;C:/Qt/5.0.1/msvc2012" -D CMAKE_INSTALL_PREFIX="./install" -D BR_INSTALL_DEPENDENCIES=ON -D CMAKE_BUILD_TYPE=Release ..
      $ nmake
      $ nmake install
    2. Check out the "install" folder.
  7. Hack OpenBR!
    1. Download Qt Creator IDE and install.
    2. From the VS2012 x64 Cross Tools Command Prompt:
      $ C:\Qt\qtcreator-2.6.2\bin\qtcreator.exe
    3. From the Qt Creator "Tools" menu select "Options..."
    4. Under "Kits" select "Desktop (default)"
    5. For "Compiler:" select "Microsoft Visual C++ Compiler 11.0 (amd64)" and click "OK"
    6. From the Qt Creator "File" menu select "Open File or Project...".
    7. Select "C:\openbr\CMakeLists.txt" then "Open".
    8. Browse to your prexisting build directory "C:\openbr\build-msvc2012" then select "Next".
    9. Clear any text in the "arguments" box then select "Run CMake" then "Finish".
    10. You're all set! You can find more information on Qt Creator here if you need.
  8. Package OpenBR!
    1. Download NSIS 2.46 and install.
    2. From the VS2012 x64 Cross Tools Command Prompt:
      $ cd C:\openbr\build-msvc2012
      $ nmake package
