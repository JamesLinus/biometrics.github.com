  • Installation
Windows 7 - MinGW-w64 2.0 - x64
  1. Download and Unarchive MinGW-w64 GCC 4.7.2
    1. Use the free open source program 7-Zip to unarchive.
    2. Move "x86_64-w64-mingw32-gcc-4.7.2-release-win64_rubenvb\mingw64" to "C:\".
  2. Download and Install CMake 2.8.10.2
    1. During installation setup select "add CMake to PATH".
  3. Download OpenCV 2.4.5
    1. Consider the free open source program 7-Zip if you need a program to unarchive tarballs.
    2. Move the "opencv-2.4.5" folder to "C:\".
    3. From the MinGW-w64 Command Prompt (double-click "C:\mingw64\mingw64env.cmd"):
      $ cd C:\opencv-2.4.5
      $ mkdir build-mingw64
      $ cd build-mingw64
      $ cmake -G "MinGW Makefiles" -DBUILD_PERF_TESTS=OFF -DBUILD_TESTS=OFF -DWITH_FFMPEG=OFF -DCMAKE_BUILD_TYPE=Debug ..
      $ mingw32-make
      $ mingw32-make install
      $ cmake -DCMAKE_BUILD_TYPE=Release ..
      $ mingw32-make
      $ mingw32-make install
      $ mingw32-make clean
  4. Download and Unzip Qt 5.0.2
    1. Install Perl/Python/Ruby dependencies as explained in the "Windows" section of "README". Make sure they are added to "path" when given the option during installation.
    2. Download and Install Direct X Software Developement Kit, you may also need to install the latest OpenGL drivers from your graphics card manufacturer.
    3. From the MinGW-w64 Command Prompt:
      $ cd qt-everywhere-opensource-src-5.0.2
      $ configure -prefix C:\Qt\Qt5.0.2\5.0.2\mingw64 -opensource -confirm-license -nomake examples -nomake tests -opengl desktop
      $ mingw32-make
      $ mingw32-make install
      $ cd ..
      $ rmdir /Q /S qt-everywhere-opensource-src-5.0.2
    4. mingw32-make will take several hours to finish.
  5. Create a GitHub account and follow their instructions for setting up Git.
    1. Launch "Git Bash" from the Desktop and clone OpenBR:
      $ cd /c
      $ git clone https://github.com/biometrics/openbr.git
      $ cd openbr
      $ git submodule init
      $ git submodule update
  6. Build OpenBR!
    1. From the MinGW-w64 Command Prompt:
      $ cd C:\openbr
      $ mkdir build-mingw64
      $ cd build-mingw64
      $ cmake -G "CodeBlocks - MinGW Makefiles" -DCMAKE_RC_COMPILER="C:/mingw64/bin/windres.exe" -DCMAKE_PREFIX_PATH="C:/opencv-2.4.5/build-mingw64/install;C:/Qt/Qt5.0.2/5.0.2/mingw64" -DCMAKE_INSTALL_PREFIX="./install" -DBR_INSTALL_DEPENDENCIES=ON -DCMAKE_BUILD_TYPE=Release ..
      $ mingw32-make
      $ mingw32-make install
    2. Check out the "install" folder.
  7. Hack OpenBR!
    1. From the MinGW-w64 Command Prompt:
      $ C:\Qt\Qt5.0.2\Tools\QtCreator\bin\qtcreator.exe
    2. From the Qt Creator "Tools" menu select "Options..."
    3. Under "Kits" select "Desktop (default)"
    4. For "Compiler:" select "MinGW (x86 64bit in C:\mingw64\bin)" and click "OK"
    5. From the Qt Creator "File" menu select "Open File or Project...".
    6. Select "C:\openbr\CMakeLists.txt" then "Open".
    7. If prompted for the location of CMake, enter "C:\Program Files (x86)\CMake 2.8\bin\cmake.exe".
    8. Browse to your pre-existing build directory "C:\openbr\build-mingw64" then select "Next".
    9. Clear any text in the "arguments" box then select "Run CMake" then "Finish".
    10. You're all set! You can find more information on Qt Creator here if you need.
  8. (Optional) Package OpenBR!
    1. From the MinGW-w64 Command Prompt:
      $ cd C:\openbr\build-mingw64
      $ cpack -G ZIP
