  • Installation
Ubuntu 12.04 LTS - Intel C++ Studio XE 2013 - x64
  1. Assuming you meet the eligibility requirements, Download Intel C++ Studio XE 2013 and install.
  2. Download CMake 2.8.10.2.
    $ cd ~/Downloads
    $ tar -xf cmake-2.8.10.2.tar.gz
    $ cd cmake-2.8.10.2
    $ ./configure
    $ make -j4
    $ sudo make install
    $ cd ..
    $ rm -r cmake-2.8.10.2
  3. Download OpenCV 2.4.4.
    $ cd ~/Downloads
    $ tar -xf OpenCV-2.4.4.tar.bz2
    $ cd OpenCV-2.4.4
    $ mkdir build
    $ cd build
    $ cmake cmake -DBUILD_opencv_java=OFF -DBUILD_opencv_world=ON -DCMAKE_BUILD_TYPE=Release ..
    $ make -j4
    $ sudo make install
    $ cd ../..
    $ rm -r OpenCV-2.4.4
  4. Download Qt 5.0.1.
    $ cd ~/Downloads
    $ chmod +x qt-linux-opensource-5.0.1-x86_64-offline.run
    $ ./qt-linux-opensource-5.0.1-x86_64-offline.run
    $ rm qt-linux-opensource-5.0.1-x86_64-offline.run
  5. Create a GitHub account, follow their instructions for setting up Git.
    $ git clone https://github.com/biometrics/openbr.git
    $ cd openbr
    $ git submodule init
    $ git submodule update
  6. Build OpenBR!
    $ cd openbr
    $ mkdir build-icc
    $ cd build-icc
    $ cmake -DCMAKE_C_COMPILER=/opt/intel/bin/icc -DCMAKE_CXX_COMPILER=/opt/intel/bin/icpc -DCMAKE_PREFIX_PATH=~/Qt5.0.1/5.0.1/gcc_64 -DCMAKE_BUILD_TYPE=Release ..
    $ make -j4
    $ sudo make install
  7. Hack OpenBR!
    1. Open Qt Creator IDE
      $ ~/Qt5.0.1/Tools/QtCreator/bin/qtcreator &
    2. From the Qt Creator "File" menu select "Open File or Project...".
    3. Select "~/openbr/CMakeLists.txt" then "Open".
    4. Browse to your prexisting build directory "~/openbr/build" then select "Next".
    5. Select "Run CMake" then "Finish".
    6. You're all set! You can find more information on Qt Creator here if you need.
  8. (Optional) Package OpenBR!
    $ cd openbr/build
    $ make package
