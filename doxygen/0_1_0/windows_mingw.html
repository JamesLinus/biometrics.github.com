Windows 7 - MingGW-w64 2.0 - x64

Installation from source with MinGW-w64.

  1. Download MinGW-w64 GCC 4.7.2 and unarchive.
    1. Use the free open source program 7-Zip to unarchive.
    2. Copy "x86_64-w64-mingw32-gcc-4.7.2-release-win64_rubenvb\mingw64" to "C:\".
  2. Download CMake 2.8.10.2 and install.
    1. During installation setup select "add CMake to PATH".
  3. Download OpenCV 2.4.3 and unarchive.
    1. If you need a program to unarchive tarballs then consider the free open source program 7-Zip.
    2. Copy the "OpenCV-2.4.3" folder to "C:\" and rename it "OpenCV-2.4.3-mingw64".
    3. Double-click "C:\mingw64\mingw64env.cmd" and enter:
      $ cd C:\OpenCV-2.4.3-mingw64
      $ mkdir build
      $ cd build
      $ cmake -G "MinGW Makefiles" -D WITH_FFMPEG=OFF -D CMAKE_BUILD_TYPE=Release ..
      $ mingw32-make
  4. Download Qt 4.8.4 and unzip.
    1. Copy "qt-everywhere-opensource-src-4.8.4" to "C:\".
    2. Double-click "C:\mingw64\mingw64env.cmd" and enter:
      $ cd C:\qt-everywhere-opensource-src-4.8.4
      $ configure.exe
      $ mingw32-make
      1. Select the Open Source Edition.
      2. Accept the license offer.
      3. configure.exe will take several minutes to finish.
      4. mingw32-make will take several hours to finish.
  5. Create a GitHub account and follow their instructions for setting up Git.
    1. Launch "Git Bash" and clone OpenBR:
      $ cd /c
      $ git clone https://github.com/biometrics/openbr.git
      $ cd openbr
      $ git submodule init
      $ git submodule update
  6. Finally time to build OpenBR:
    1. Double-click "C:\mingw64\mingw64env.cmd" and enter:
      $ cd C:\openbr
      $ mkdir build-mingw64
      $ cd build-mingw64
      $ cmake -G "MinGW Makefiles" -D CMAKE_RC_COMPILER="C:/mingw64/bin/windres.exe" -D OpenCV_DIR="C:\OpenCV-2.4.3-mingw64\build" -D QT_QMAKE_EXECUTABLE="C:\qt-everywhere-opensource-src-4.8.4\bin\qmake" ..
      $ mingw32-make
  7. To package OpenBR:
    1. Download NSIS 2.46 and install.
    2. Double-click "C:\mingw64\mingw64env.cmd" and enter:
      $ cd C:\openbr\build-mingw64
      $ mingw32-make package
