About

OpenBR was developed as a MITRE internal research project to facilitate prototyping new biometric algorithms and evaluating commercial systems. It has been open sourced with the hope of providing a common framework for algorithm development and evaluation.

OpenBR is written entirely in C/C++ and follows the Semantic Versioning convention for publishing releases. The project uses the CMake build system and depends on Qt 4.8 and OpenCV 2.4.3. The Biometric Evaluation Environment and the conventions established in the MBGC File Overview for experimental setup are used for evaluating algorithm performance.

  • Developer mailing list: openbr-dev at googlegroups.com
  • IRC channel: openbr
  • Continuous integration server: CDash
