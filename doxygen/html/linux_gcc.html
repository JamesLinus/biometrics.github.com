Ubuntu 12.04 LTS - GCC 4.6.3 - x64
  1. Install GCC 4.6.3:
    $ sudo apt-get update
    $ sudo apt-get install build-essential
  2. Install CMake 2.8.7:
    $ sudo apt-get install cmake
  3. Download OpenCV 2.4.3 and install:
    $ cd ~/Downloads
    $ tar -xf OpenCV-2.4.3.tar.bz2
    $ cd OpenCV-2.4.3
    $ mkdir build
    $ cd build
    $ cmake -D CMAKE_BUILD_TYPE=Release ..
    $ make
    $ sudo make install
  4. Install Qt 4.8.1:
    $ sudo apt-get install libqt4-dev
  5. Create a GitHub account, follow their instructions for setting up Git, then clone:
    $ git clone https://github.com/biometrics/openbr.git
  6. Finally time to build OpenBR:
    $ cd mm
    $ mkdir build
    $ cd build
    $ cmake ..
    $ make
  7. To package OpenBR:
    $ cd mm/build
    $ make package
