Windows 7 - Visual Studio Express Edition 2012 - x64

Installation from source with Visual Studio.

  1. Download and install Visual Studio 2012 Express Edition for Windows Desktop
    1. If you need a program to mount ISO images then consider the free open source program WinCDEmu.
    2. You will have to register with Microsoft after installation, but it's free.
    3. Grab any available Visual Studio Updates.
  2. Download and install CMake 2.8.10.2
    1. During installation setup select "add CMake to PATH".
  3. Download and unarchive OpenCV 2.4.3
    1. If you need a program to unarchive tarballs then consider the free open source program 7-Zip.
    2. Copy the "OpenCV-2.4.3" folder to "C:\" and rename it "OpenCV-2.4.3-msvc2012".
    3. From the Start Menu, select "All Programs" -> "Microsoft Visual Studio 2012" -> "Visual Studio Tools" -> "VS2012 x64 Cross Tools Command Prompt" and enter:
      $ cd C:\OpenCV-2.4.3-msvc2012
      $ mkdir build
      $ cd build
      $ cmake -G "Visual Studio 11 Win64" -D WITH_FFMPEG=OFF ..
    4. Open "C:\OpenCV-2.4.3-msvc2012\build\OpenCV.sln"
      1. Under the "BUILD" menu, select "Build Solution".
      2. Switch from "Debug" to "Release" and repeat the above step.
  4. Download and install Qt 4.8.4
    1. From the Start Menu, select "All Programs" -> "Microsoft Visual Studio 2012" -> "Visual Studio Tools" -> "VS2012 x64 Cross Tools Command Prompt" and enter:
      $ cd C:\Qt\4.8.4
      $ configure.exe -platform win32-msvc2012 -no-webkit
      $ nmake
      1. Select the Open Source Edition.
      2. Accept the license offer.
      3. configure.exe will take several minutes to finish.
      4. nmake will take several hours to finish.
  5. Create a GitHub account and follow their instructions for setting up Git.
    1. Launch "Git Bash" and clone OpenBR:
      $ cd /c
      $ git clone https://github.com/biometrics/openbr.git
      $ cd openbr
      $ git submodule init
      $ git submodule update
  6. Finally time to actually build OpenBR:
    1. From the Start Menu, select "All Programs" -> "Microsoft Visual Studio 2012" -> "Visual Studio Tools" -> "VS2012 x64 Cross Tools Command Prompt" and enter:
      $ cd C:\openbr
      $ mkdir build-msvc2012
      $ cd build-msvc2012
      $ cmake -G "NMake Makefiles" -D OpenCV_DIR="C:/OpenCV-2.4.3-msvc2012/build" -D QT_QMAKE_EXECUTABLE="C:/Qt/4.8.4/bin/qmake" ..
      $ nmake
  7. To package OpenBR:
    1. From the Start Menu, select "All Programs" -> "Microsoft Visual Studio 2012" -> "Visual Studio Tools" -> "VS2012 x64 Cross Tools Command Prompt" and enter:
      $ cd C:\openbr\build-msvc2012
      $ nmake package
